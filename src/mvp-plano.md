## Plano de MVP
- Funcionalidades principais:
  1. Chat para registro de gastos.
  2. Classificação automática.
  3. Metas financeiras.
  4. Dicas do agente IA.
  5. Relatórios simples.

- Recursos necessários:
  - Frontend: React Native.
  - Backend: Node.js.
  - IA: API OpenAI ou Copilot.

- Validação inicial:
  - Teste com 10 usuários.
  - Métrica: clareza da conversa e utilidade das dicas.
